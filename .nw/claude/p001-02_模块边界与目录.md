## 一、目录结构

```
vue/src/
├── main.ts                       # 入口：引导流程 → mount
├── App.vue                       # <RouterView/> + <LoaderBar/>
├── styles/                       # variables.scss / index.scss / transitions.scss
├── types/                        # abp.ts / api.ts / identity.ts / tenant.ts / permission.ts / feature.ts / settings.ts
├── config/env.ts                 # API 基址、OAuth 配置
├── router/
│   ├── index.ts                  # 路由树
│   └── guards/                   # auth.ts / permission.ts
├── stores/                       # app-config.ts / auth.ts / session.ts
├── api/                          # http.ts + 7 模块代理文件（纯函数薄封装）
├── composables/                  # useAuth / usePermission / useLocalization / useTenant
├── plugins/                      # i18n.ts / oidc.ts
├── directives/                   # v-permission.ts
├── components/                   # AbpDataTable / AbpModal / AbpDynamicForm / AbpToast / AbpBreadcrumb / AbpLoaderBar / AbpEmptyState / AbpErrorPage / AbpConfirmDialog
├── layouts/
│   ├── ApplicationLayout.vue     # 侧边菜单 + 顶栏 + 面包屑 + 主内容
│   ├── AccountLayout.vue         # 居中认证框
│   ├── EmptyLayout.vue           # 纯内容
│   └── partials/                 # SideMenu / TopBar / UserMenu / LangSwitch / TenantBox
├── views/
│   ├── auth/                     # Login / Register / ForgotPassword / ResetPassword / ManageProfile
│   ├── identity/                 # UsersView / RolesView + components/
│   ├── tenant/                   # TenantsView + components/
│   └── settings/                 # SettingsView + EmailSettingsTab + TimezoneSettingsTab
└── utils/                        # abp-error.ts / permission-modal.ts / feature-modal.ts
```

## 二、模块边界合约

### 数据层（stores/api）

| 模块 | 拥有 | 导出 | 不拥有 |
|------|------|------|--------|
| `api/http.ts` | Axios 实例、5 拦截器 | `httpClient` | Token 存储 |
| `stores/app-config.ts` | `/application-configuration` 缓存 | `useAppConfigStore()` | 写入 |
| `stores/auth.ts` | OIDC user/token（持久化） | `useAuthStore()` | 登录 UI |
| `stores/session.ts` | 语言/租户偏好（持久化） | `useSessionStore()` | — |
| `api/*.ts` | API 端点纯函数 | 按模块分组函数 | 状态/缓存 |

### 基础设施层（plugins/composables/directives）

| 模块 | 拥有 | 导出 | 不拥有 |
|------|------|------|--------|
| `plugins/i18n.ts` | vue-i18n 实例、远程加载 | `$t()`, `useLocalization()` | — |
| `composables/usePermission.ts` | 权限检查 | `usePermission()`, `permissionGuard` | 权限管理 UI |
| `directives/v-permission.ts` | DOM 级权限控制 | `v-permission` | — |
| `plugins/oidc.ts` | oidc-client-ts 配置 | — | 登录 UI |
| `composables/useAuth.ts` | login/logout/getToken | `useAuth()` | — |

### 表现层（components/layouts/views/utils）

| 模块 | 拥有 | 导出 | 不拥有 |
|------|------|------|--------|
| `components/` | DataTable/Modal/DynamicForm/Toast 等 | 各组件/函数 | 业务逻辑 |
| `layouts/` | 3 种布局 + 菜单/顶栏/面包屑 | 布局组件 | 页面内容 |
| `views/auth/` | 5 个认证页面 | 路由配置 | Token 管理 |
| `views/identity/` | 用户/角色列表+CRUD 模态框 | 路由配置 | 权限管理 UI |
| `views/tenant/` | 租户列表+CRUD+连接字符串 | 路由配置 | 功能管理 UI |
| `views/settings/` | 标签页容器+邮件+时区 | 路由配置 | — |
| `utils/permission-modal.ts` | 权限树+资源权限 | `openPermissionModal()` → `{open,close,onSaved}` | 自己的路由 |
| `utils/feature-modal.ts` | 功能组+值类型+级联 | `openFeatureModal()` → `{open,close,onSaved}` | 自己的路由 |
