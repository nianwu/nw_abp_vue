## Context

项目尚无 Vue 代码，需以 `demo/` Angular 应用为功能参考，在 Vue 3 中落地 ABP 默认模板的全部必要性功能。本规划聚焦"如何实施"——任务拆分、优先级排序、并行策略、验收标准。功能需求详见 `docs/S006-Vue端功能需求清单.md`。

## 开发流程：「垂直切片 → 水平扩展」

```
Phase A（P0，顺序）                      Phase B（P1-P3，全部可并行）

A1 脚手架 → A2 API+HTTP+类型              B1 [P1] 角色管理
  → A3 Pinia → A4 i18n+权限+OIDC         B2 [P1] 个人资料+认证页
  → A5 UI组件定型 → A6 布局               B3 [P2] 租户管理
  → A7 登录+用户管理（端到端验证）          B4 [P2] 权限管理（对象式）
                                          B5 [P2] 功能管理（对象式）
✅ 架构验证 → Phase B 全部可并行启动        B6 [P3] 设置管理
                                          B7 [P3] 集成收尾
```
