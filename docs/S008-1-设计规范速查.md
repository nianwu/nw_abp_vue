## 一、表格操作列 — Link 样式

所有表格页操作按钮统一使用 Element Plus `link` 样式 + `size="small"` + 图标：

```html
<el-button size="small" link type="primary" :icon="Edit">编辑</el-button>
<el-button size="small" link type="danger"  :icon="Delete">删除</el-button>
<el-button size="small" link type="warning" :icon="Lock">权限</el-button>
<el-button size="small" link type="success" :icon="Connection">连接字符串</el-button>
<el-button size="small" link type="warning" :icon="Setting">功能</el-button>
```

**颜色约定**：编辑=primary · 删除=danger · 权限/功能类=warning

操作列 CSS：`display: flex; flex-wrap: nowrap; gap: 4px; justify-content: center`，按钮不换行紧凑排列。

## 二、弹窗 — 命令式 API

PermissionModal 和 FeatureModal 不通过模板声明，改用命令式创建：

```ts
import { openPermissionModal } from '@/utils/permission-modal'

const modal = openPermissionModal({ providerName: 'R', providerKey: 'admin' })
modal.onSaved(() => { /* 刷新列表 */ })
modal.open()
```

- 组件实例挂载到 `document.body`，关闭后自动销毁并移除 DOM。
- 宿主页面无需在模板中写 `<PermissionModal ... />`。
- 同理 `openFeatureModal()` 用于功能管理弹窗。

## 三、时间字段 — 三种渲染模式

所有日期列使用 `DateTimeCell` 组件，支持三种模式：

| 模式 | 示例 | 说明 |
|------|------|------|
| `full` | `2026/7/1 14:30:00` | 完整日期 |
| `relative` | `3 小时前` | 相对时间，超过 3 天回退完整日期 |
| `combined` | `2026/7/1 14:30:00 (3 小时前)` | 完整+相对 |

- `relative` 模式鼠标悬停显示完整日期（tooltip）。
- 列配置中加 `dateRender: true` 即可启用，用户可在列设置面板中切换模式。
- 底层基于 `date-fns` + `zhCN` locale。

## 四、表格列设置面板

`AbpDataTable` 工具栏右侧齿轮图标 → `el-popover` 列设置面板，包含：

1. **列可见性**：checkbox 勾选显示/隐藏
2. **日期渲染模式**：标记了 `dateRender` 的列出现 radio-group 切换三种模式
3. **列宽管理**：拖拽列宽自动持久化到 localStorage，可逐列重置或一键重置全部
4. **分页大小**：通过 `storageKey` prop 持久化到 localStorage

localStorage key 命名规则：`abp:<功能>:<storageKey>:<prop>`

## 五、非大众词汇 — 帮助入口约定

ABP 中"资源权限""资源密钥"等概念对非 ABP 开发者不直观，项目中遵循以下约定：

| 场景 | 方式 | 示例 |
|------|------|------|
| **概念性帮助** | 独立说明页（新标签页 `window.open`） | `/docs/resource-permissions`、`/docs/resource-key` |
| **操作性提示** | `el-tooltip` 悬浮提示 | "已通过配置角色获取该权限" |

入口：`PermissionModal` 中 Tab 标签旁和 label 旁放置 `QuestionFilled` 图标，点击跳转说明页。

## 六、Provider Key — 单字母缩写

在权限管理和功能管理中，Provider 类型使用单字母缩写：

| 缩写 | 含义 | 标签颜色 |
|------|------|---------|
| `R` | Role（角色） | primary (蓝) |
| `U` | User（用户） | success (绿) |
| `T` | Tenant（租户） | warning (橙) |

---

## 七、表格排序箭头 — CSS 脱离文档流

Element Plus 排序箭头默认占据列宽。项目中统一用 CSS 将其设为 `position: absolute`，使列头文字居中时不受箭头宽度影响：

```css
.el-table__header-wrapper .caret-wrapper {
  position: absolute;
  right: 2px;
  top: 50%;
  transform: translateY(-50%);
}
```

## 八、弹窗内 Tab 表单

复杂实体的创建/编辑弹窗使用 `el-tabs` 拆分为多个 Tab（如 `UserCreateEditModal`：基本信息 + 角色分配），避免单一长表单。
