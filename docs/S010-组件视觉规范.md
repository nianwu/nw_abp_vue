## 一、AbpDataTable（数据表格）

**标准结构**（自上而下）：

```
[搜索框] [新建按钮]              [列可见性切换]
────────────────────────────────────────────
[表头]
[数据行]（stripe 斑马纹 + border 竖线）
[空状态 / 骨架屏]
────────────────────────────────────────────
                             [分页器（右对齐）]
```

### 要求

- 表格属性：`stripe border`（必须）
- 操作列：固定右侧（`fixed="right"`），宽 `180px`
- 操作按钮尺寸：统一 `size="small"`
- 删除按钮：`type="danger"`；静态/系统内置项设 `disabled`
- 搜索框：`w-56`（224px），前缀搜索图标，防抖 300ms
- 首次加载：`el-skeleton :rows="5" animated`
- 后续刷新：表格 `v-loading`
- 空状态：`AbpEmptyState`，区分 `empty` / `no-results`
- 分页器：`layout="total, sizes, prev, pager, next, jumper"`，右对齐
- 日期格式化：`yyyy-MM-dd HH:mm:ss`，空值 `-`
- 列可见性弹窗触发按钮仅显示齿轮图标，须添加 tooltip "列设置"

## 二、AbpModal（弹窗）

### 尺寸

| 尺寸 | 宽度 | 适用场景 |
|------|------|---------|
| `sm` | 420px | 简单确认 |
| `md` | 600px | 创建/编辑（默认） |
| `lg` | 800px | 功能管理、连接字符串 |
| `xl` | 1000px | 权限管理（含左侧标签页） |
| — | 100% | 移动端 <768px（自动全屏） |

### 要求

- `destroy-on-close` — 每次打开重新初始化（必须）
- `:close-on-click-modal="false"` — 防误触（必须）
- 有未保存更改（`dirty=true`）时关闭前二次确认
- 底部：取消（左，默认样式）+ 确认（右，`type="primary"`）
- 内部间距：字段间 `mb-4`，分组间 `mb-6`

## 三、AbpDynamicForm（动态表单）

### 支持类型

| 类型 | 渲染组件 |
|------|---------|
| `text` | `el-input` |
| `number` | `el-input-number` |
| `email` | `el-input type="email"` |
| `password` | `el-input type="password" show-password` |
| `switch` | `el-switch` |
| `select` | `el-select` + `el-option` |
| `date` | `el-date-picker type="date" value-format="YYYY-MM-DD"` |
| `textarea` | `el-input type="textarea" :rows="3"` |

### 要求

- 全局 `label-position="top"`
- 必填字段红色星号（`required` prop）
- 验证错误显示在字段下方
- Select / DatePicker 使用 `class="w-full"`
- Switch 标签与控件同行，标签左控件右

---

## 四、AbpBreadcrumb（面包屑）

- 分隔符：`/`
- 垂直内边距：`py-2`（8px）
- 末项不可点击，上级可点击导航
- 标题取 `route.meta.title`，回退 `route.name`

---

## 五、AbpEmptyState（空状态）

| 类型 | 描述 | 可带操作 |
|------|------|---------|
| `empty` | 暂无数据 | "新建"按钮 |
| `no-results` | 未找到匹配结果 | 无 |
| `no-options` | 暂无可用选项 | 无 |

### 要求

- 垂直居中，`py-12`
- 图标 `120px`
- 按钮仅 `type="empty"` 时可选显示

---

## 六、AbpErrorPage（错误页）

| 错误码 | 标题 | 描述 | 操作按钮 |
|--------|------|------|---------|
| 403 | 无权限访问 | 您没有访问此页面的权限 | 返回上一页 |
| 404 | 页面未找到 | 您访问的页面不存在 | 返回首页 |
| 500 | 服务器错误 | 服务器发生异常，请稍后重试 | **返回首页（当前缺失，须补充）** |

### 要求

- 居中布局
- 错误码：`text-6xl font-bold text-gray-300`
- 标题：`text-xl text-gray-600`
- 描述：`text-gray-400`
- 所有错误码均须有导航出口

---

## 七、PermissionModal（权限管理弹窗）

### 要求

- 弹窗宽度 900px
- 左侧 `el-tabs` 切换权限组，每组内 `el-tree` 显示权限树
- 权限树 `default-expand-all`，节点 `font-size: 13px`
- 授权来源徽章：`size="small"`，`font-size: 11px`
- 更改摘要固定在底部边框上方：授权绿 + 回收红
- 资源权限 Tab：列表 → 添加 → 编辑 三视图切换
- 保存/取消前校验未保存更改

---

## 八、FeatureModal（功能管理弹窗）

### 要求

- 弹窗宽度 800px
- 按功能组分 Tab，每组内 `el-tree` 显示功能树
- 功能值类型决定控件：
  - `ToggleStringValueType` → `el-switch`
  - `FreeTextStringValueType` → `el-input size="small" width: 220px`
  - `SelectionStringValueType` → `el-select size="small" width: 220px`
  - 其他 → `el-tag` 只读显示
- 父功能关闭时子功能自动禁用（级联 `_disabled`）
- 更改数量提示（橙色 `--el-color-warning`）
- 保存/取消前校验未保存更改

---

## 九、TopBar 内嵌组件

### TenantBox

- 仅多租户启用时渲染（`v-if`）
- 显示当前租户名（默认 "Host"）
- `text-white text-sm`

### LangSwitch

- `el-dropdown` 触发，按钮 `el-button link text-white`
- 当前语言显示 `displayName`

### UserMenu

- 头像 28px（`el-avatar icon="UserFilled"`）+ 用户名
- 小屏（<640px）：隐藏用户名，仅头像
- 下拉项：个人资料 → 退出登录（分割线分隔）

---

## 十、NProgress（全局加载条）

- 颜色：`#409EFF`（`--el-color-primary`）
- 高度：3px
- 动画：`#nprogress .peg` 带 `box-shadow` 光晕
- 通过 `AbpLoaderBar` 组件 + HTTP 拦截器自动联动
