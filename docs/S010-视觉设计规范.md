## 一、设计原则

| 原则 | 说明 |
|------|------|
| **清晰优先** | 信息层级分明，操作意图明确。不使用装饰性元素干扰数据阅读 |
| **一致即效率** | 同类元素在间距、色彩、交互反馈上保持统一，降低认知负荷 |
| **轻量企业风** | 低饱和度配色 + 适度留白，避免"重管理后台"的压抑感 |
| **渐进呈现** | 优先展示核心数据，详情与高级操作通过弹窗/展开渐进披露 |
| **触达即用** | 移动端不做阉割，通过抽屉、折叠等模式保证功能可访问 |

## 二、色彩系统

### 2.1 语义令牌

项目使用 Element Plus CSS 变量体系，所有组件色值须通过 CSS 变量引用，**禁止硬编码颜色值**。

| 令牌 | 色值 | 用途 |
|------|------|------|
| `--el-color-primary` | `#409EFF` | 主操作按钮、激活态、链接 |
| `--el-color-success` | `#67C23A` | 成功状态、启用标签 |
| `--el-color-warning` | `#E6A23C` | 警告状态、静态标签 |
| `--el-color-danger` | `#F56C6C` | 危险操作、删除按钮 |
| `--el-color-info` | `#909399` | 次要信息、禁用态文本 |

### 2.2 布局色彩

| 令牌 | 色值 | 用途 |
|------|------|------|
| `--el-menu-bg-color` | `#304156` | 侧栏 & 顶栏背景 |
| `--el-menu-text-color` | `#BFCBD9` | 侧栏菜单文字 |
| `--el-menu-active-color` | `#409EFF` | 侧栏菜单激活项 |
| `--el-bg-color-page` | `#F5F7FA` | 页面底色 |
| `--el-bg-color` | `#FFFFFF` | 卡片/表格/弹窗背景 |
| `--el-text-color-primary` | `#303133` | 正文 |
| `--el-text-color-regular` | `#606266` | 常规文字 |
| `--el-text-color-secondary` | `#909399` | 辅助说明文字 |
| `--el-border-color` | `#DCDFE6` | 边框 |
| `--el-border-color-light` | `#E4E7ED` | 浅边框（分割线） |

### 2.3 权限来源色映射

权限弹窗中不同提供者使用差异化颜色：

| 提供者 | 标签色 | 含义 |
|--------|--------|------|
| `R` | `primary` (蓝) | Role 授予 |
| `U` | `success` (绿) | User 授予 |
| `T` | `warning` (橙) | Tenant 授予 |

## 三、字体系统

### 3.1 字体族

```css
font-family: -apple-system, BlinkMacSystemFont, 'Segoe UI', Roboto,
  'Helvetica Neue', Arial, 'Noto Sans SC', sans-serif;
```

西文优先系统原生无衬线栈，中文使用 Noto Sans SC（思源黑体）。

### 3.2 字号层级

| 层级 | Tailwind | 字号 | 行高 | 用途 |
|------|---------|------|------|------|
| H1 | `text-2xl` | 24px | 32px | 页面标题 |
| H2 | `text-xl` | 20px | 28px | 卡片/认证页标题 |
| H3 | `text-lg` | 18px | 26px | 分组标题（预留） |
| Body | — | 14px | 22px | 正文、表格、表单标签 |
| Body-Small | `text-sm` | 13px | 20px | 辅助信息、标签、徽章 |
| Caption | `text-xs` | 12px | 18px | 提示、大写锁定警告 |

### 3.3 字重

| 用途 | 字重 | Tailwind |
|------|------|---------|
| 页面标题 | 700 | `font-bold` |
| 区块标题 | 600 | `font-semibold` |
| 表单标签 | 500 | Element Plus 默认 |
| 正文 | 400 | 默认 |

## 四、间距系统

### 4.1 间距量表（Tailwind 4px 基准）

| Token | 值 | 用途 |
|-------|-----|------|
| `p-2` / `gap-2` | 8px | 紧凑间距（按钮组、标签间隙） |
| `p-3` / `gap-3` | 12px | TopBar 元素间距 |
| `p-4` / `gap-4` | 16px | **页面统一外边距** |
| `mb-4` | 16px | 区块下间距 |
| `mb-6` | 24px | 大区块下间距（标题与内容） |
| `mb-8` | 32px | 页面级下间距（Logo 与卡片） |

### 4.2 全局规则

- 管理页根元素统一 `class="p-4"`
- 表格搜索栏与表格间距 `mb-4`
- 表格与分页间距 `mt-4`
- 表单字段间距由 Element Plus `el-form-item` 默认控制

### 4.3 布局常量

| 常量 | 值 | Sass 变量 |
|------|-----|-----------|
| 侧栏展开宽度 | 260px | `$sidebar-width` |
| 侧栏折叠宽度 | 64px | `$sidebar-collapsed-width` |
| 顶栏高度 | 48px | `$topbar-height` |
| 面包屑高度 | 36px | `$breadcrumb-height` |

**要求**：`ApplicationLayout.vue` 中宽度值须引用变量，不与 TS 中硬编码耦合。

## 五、布局架构

### 5.1 三层布局

| 布局 | 适用场景 | 结构 |
|------|---------|------|
| **ApplicationLayout** | 管理页 | 侧栏（左）+ 顶栏 + 内容区（面包屑 + 页面） |
| **AccountLayout** | 认证页 | 全屏居中卡片（`max-w-md`，`bg-gray-100`） |
| **EmptyLayout** | OIDC/错误页 | 直接 `<router-view />` |

### 5.2 ApplicationLayout 结构

```
┌──────────┬──────────────────────────────┐
│          │ TopBar (48px, --el-menu-bg)   │
│ Sidebar  ├──────────────────────────────┤
│ (260/64) │ Breadcrumb                    │
│          ├──────────────────────────────┤
│          │ Page Content (p-4)            │
└──────────┴──────────────────────────────┘
```

**要求**：
- 侧栏折叠/展开时 `el-aside` 和 `el-main` 同步过渡（`transition: 0.3s`）
- 移动端（<768px）侧栏改为 `el-drawer` 左滑出，方向 `ltr`
- 顶栏高度固定 48px，内容垂直居中

### 5.3 响应式断点

| 断点 | 宽度 | 布局变化 |
|------|------|---------|
| `sm` | ≥576px | 基础 |
| `md` | ≥768px | 侧栏可见，抽屉隐藏 |
| `lg` | ≥992px | 侧栏 260px |
| `xl` | ≥1200px | 全功能 |

JS 中 `window.innerWidth` 判断须与 Tailwind `screens` 配置保持一致。
