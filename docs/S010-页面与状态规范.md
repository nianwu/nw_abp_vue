# S010-页面与状态规范

> 主文档：[S010-视觉设计规范](./S010-视觉设计规范.md)

## 一、页面模板

### 1.1 标准列表管理页

适用于 UsersView / RolesView / TenantsView。

```html
<div class="p-4">
  <AbpDataTable
    search-placeholder="搜索XXX..."
    :columns="[...]"
    :api="fetchItems"
  >
    <template #toolbar-actions>
      <el-button type="primary" @click="openCreate">新建XXX</el-button>
    </template>
    <template #actions="{ row }">
      <!-- 编辑 / 权限 / 删除 -->
    </template>
  </AbpDataTable>
  <CreateEditModal v-model:visible="..." :item-id="..." @saved="refresh" />
</div>
```

### 1.2 设置页

```html
<div class="p-4">
  <h2 class="text-xl mb-6">设置管理</h2>
  <el-tabs type="border-card">
    <el-tab-pane label="标签1" lazy>...</el-tab-pane>
    <el-tab-pane label="标签2" lazy>...</el-tab-pane>
  </el-tabs>
</div>
```

### 1.3 认证页

由 `AccountLayout` 统一包裹，各页仅关注表单：

- 标题：`h2 class="text-center text-xl font-semibold mb-6"`
- 表单：`el-form label-position="top"`
- 提交按钮：`el-button type="primary" class="w-full"`
- 辅助链接：`text-sm`，`text-blue-500`，位于表单下方

### 1.4 个人资料页

双卡片（`el-card shadow="never"`），每卡片 `#header` 插槽：

- 卡片一："基本信息" — `AbpDynamicForm` + 保存
- 卡片二："修改密码" — `el-form`（当前/新密码/确认）+ 提交
- 卡片间距：`mb-6`

## 二、状态与反馈

### 2.1 加载态

| 场景 | 方式 |
|------|------|
| 页面首次加载 | 表格骨架屏（`el-skeleton :rows="5" animated`） |
| 翻页/搜索/排序 | 表格 `v-loading` |
| 弹窗数据加载 | 内容区 `v-loading` + 文字"加载中..." |
| 全局请求 | NProgress 顶部进度条（3px，#409EFF） |
| 按钮提交 | `:loading` 属性 |

### 2.2 空状态

| 场景 | 组件 |
|------|------|
| 列表无数据 | `AbpEmptyState type="empty"` + 可选"新建"按钮 |
| 搜索无结果 | `AbpEmptyState type="no-results"` |
| 下拉无选项 | `AbpEmptyState type="no-options"` |

### 2.3 错误状态

| 场景 | 方式 |
|------|------|
| API 错误 | `showError` 统一 Toast |
| 表单验证 | 字段下方红色错误（Element Plus 原生） |
| 页面级 | `AbpErrorPage`（403/404/500） |
| 模块加载失败 | 居中错误提示 + 重试按钮（`text-red-400`） |

### 2.4 成功反馈

| 场景 | 方式 |
|------|------|
| CRUD 操作 | `showSuccess` 顶部绿色消息 |
| 密码重置 | 成功消息 + 自动跳转 |
| 忘记密码 | 表单隐藏，绿色确认提示 + 返回链接 |

### 2.5 过渡动画

| 场景 | 动画 | 时长 |
|------|------|------|
| 页面切换 | fade（`opacity 0.2s ease`） | 200ms |
| 侧栏折叠 | width 过渡 | 300ms |
| 移动端侧栏 | slide-left（`transform 0.3s ease`） | 300ms |
| 弹窗 | Element Plus `el-dialog` 内置 | — |

## 三、表单规范

### 3.1 布局

- 标签位置：全局 `label-position="top"`
- 输入控件默认撑满容器宽度
- 按钮位于表单下方，左对齐

### 3.2 验证

- 必填标记：`required` prop 红色星号
- 验证时机：`trigger: 'blur'` + 提交时全量
- 错误展示：字段下方红色文字

### 3.3 特殊字段

- 密码框：`type="password" show-password`
- 登录页密码框检测 CapsLock，提示 `text-orange-500 text-xs`
- Switch：仅标签 + 开关

## 四、设计检查清单

新页面或修改现有页面时逐项核对：

- [ ] 页面根 `class="p-4"`
- [ ] 标题 `text-xl` / `text-2xl`，与内容 `mb-6`
- [ ] 色值使用 `var(--el-*)`，无硬编码
- [ ] 表格 `stripe border`，操作列 `fixed="right" width="180"`
- [ ] 弹窗 `destroy-on-close` + `:close-on-click-modal="false"`
- [ ] 表单 `label-position="top"`
- [ ] 空状态使用 `AbpEmptyState`，区分 empty / no-results
- [ ] 加载态：骨架屏 / v-loading / 按钮 loading
- [ ] 操作完成有 Toast 反馈
- [ ] 移动端抽屉导航可用
- [ ] 面包屑正确渲染
- [ ] 权限检查（操作按钮 `v-if`）
